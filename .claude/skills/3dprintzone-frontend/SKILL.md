# 3dprintzone Frontend Rules

When working on the 3dprintzone frontend:

- Create premium ecommerce UI, not generic templates.
- Match the provided reference design as closely as possible.
- Use the provided PNG assets exactly by filename and place them in the correct sections.
- Make every page fully responsive on mobile, tablet, laptop, and desktop.
- Do not use emojis anywhere in the UI.
- Use SVG icons instead of emojis.
- For category creation in admin, allow the admin to select a category logo from several SVG icon designs.
- Keep spacing, typography, animations, buttons, cards, shadows, and responsiveness polished.
- Before changing colors, ask for the exact color palette.
- When creating new folders, provide exact mkdir commands.
- When giving code changes, provide full copy-paste-ready file contents, not partial diffs.
